API cURL Examples

Use these commands to manually trigger and complete a backup operation for this configuration.

1. Create Backup Operation (POST) 
curl -X POST "http://127.0.0.1:8000/api/v1/backup-operations/" \
  -H "X-API-Key: {{ backup_configuration.target_system_version.target_system.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{
    "backup_configuration_id": {{ backup_configuration.id }},
    "hostname": "your-server-hostname",
    "ip_address": "10.0.0.1"
  }'
2. Complete Backup Operation (PATCH) 
curl -X PATCH "http://127.0.0.1:8000/api/v1/backup-operations/{operation_id}/" \
  -H "X-API-Key: {{ backup_configuration.target_system_version.target_system.api_key }}" \
  -H "Content-Type: application/json" \
  -d '{
    "status": "success",
    "size_bytes": 1048576,
    "storage_type": "local",
    "storage_path": "/path/to/backup.tar.gz"
  }'
Replace {operation_id} with the ID returned by the POST request above.
